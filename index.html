// About

Where hardware meets code

I'm a Computer Engineering undergraduate at Air University, Islamabad, currently in my 4th semester, interning as a Full Stack Intern at CapregSoft. My coursework runs from logic design and computer architecture to data structures and databases — and outside class I'm drawn to backend development, Python, FastAPI, and generative AI, building clean APIs and AI-powered prototypes.

I like systems I can trace end to end: from a gate on a breadboard, up through a datapath, into a database schema, and out through an API response. That full-stack curiosity — silicon to server — is what ties my projects together.
